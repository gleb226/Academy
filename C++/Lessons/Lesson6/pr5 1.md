## Практичне завдання: **"Міні-гра: Вгадай число!" з підказками-напрямками**

### Умова:

Створіть програму, яка моделює гру **"Вгадай число"**, але з ускладненням. Комп’ютер генерує **масив із 5 випадкових чисел** від 1 до 100. Користувач повинен вгадати хоча б одне з них, маючи **максимум 7 спроб**. Після кожної спроби програма повинна:

* повідомляти, чи є введене число серед згенерованих,
* якщо не вгадано — **підказувати**, чи найближче число в масиві **менше чи більше** за введене.

### Умови:

* Використати **масив для зберігання випадкових чисел**.
* Написати **окрему функцію для генерації випадкових чисел**.
* Написати **функцію, яка перевіряє**, чи є введене число в масиві.
* Написати **функцію, яка знаходить найближче число** в масиві до введеного, якщо не вгадано.
* Користувач має **7 спроб**. Якщо вгадав – гра завершується з перемогою. Якщо спроби закінчилися – програма виводить всі правильні числа.

---

### Приклад виконання:

```
Спроба 1. Введіть число: 50
Не вгадали. Найближче число менше за ваше.

Спроба 2. Введіть число: 46
Ви вгадали число! Вітаємо!
```

або

```
Спроба 7. Введіть число: 70
Не вгадали. Найближче число більше за ваше.

Ви не вгадали жодного з чисел.
Правильні відповіді: 18, 34, 46, 65, 89
```

---

### Підказки:

* Використовуйте `rand()` для генерації чисел, не забудьте `srand(time(0))`. (або продвинутий генератор випадкових чисел)
* Для пошуку найближчого числа — пройдіться циклом по масиву, обчислюючи модуль різниці.
* Для підказки — порівняйте введене число з найближчим і виведіть напрямок: **менше / більше**.

# Домашнє завдання: "Міні-система керування магазином"

**Опис:**  
Створити консольну програму для магазину, яка дозволяє:
- додавати товари,
- переглядати список товарів,
- конвертувати ціни,
- визначати акційні товари,
- підраховувати підсумкову вартість замовлення.

---

# Загальні вимоги:
- **Меню** зробити через `switch-case`.
- **Розгалуження** (`if-else`) для різних перевірок.
- **Типи даних**: `int`, `double`, `string`.
- **Перетворення типів**: наприклад, для роботи з кількістю/вартістю товарів або округлення.
- **Оператори**: арифметичні, логічні, порівняння.

# Структура завдань:
---

### 1. Вивести головне меню (switch-case)
При запуску користувач бачить меню:
```
1 - Додати товар
2 - Показати список товарів
3 - Конвертувати ціну товару в іншу валюту
4 - Перевірити товар на акцію
5 - Оформити замовлення
6 - Вийти
```
В залежності від вибору — переходити до відповідної функціональності.

---

### 2. Додавання товару
- Запитати назву (`string`), ціну (`double`) і кількість (`int`) товару.
- Зберегти дані у простих масивах (наприклад, три масиви: `names[]`, `prices[]`, `counts[]`). Масиви описані внизу (міні урок)
- Якщо ви ще не знайомий із масивами/векторами — можна просто обмежитись останнім введеним товаром.

---

### 3. Показати список товарів
- Вивести всі збережені товари. (міні урок теж внизу)
- Формат виводу:
  ```
  1. Хліб - 25.50 грн - 10 шт
  2. Молоко - 42.00 грн - 5 шт
  ...
  ```

---

### 4. Конвертація ціни
- Запитати номер товару зі списку.
- Запропонувати обрати валюту:
  ```
  1 - Долари
  2 - Євро
  ```
- Конвертувати ціну відповідно до фіксованого курсу валют.
- Показати нову вартість.

---

### 5. Перевірка на акційний товар
- Якщо ціна товару менша за певний поріг (наприклад, < 50 грн), вважати товар акційним і вивести відповідне повідомлення.

---

### 6. Оформити замовлення
- Дозволити користувачу вибрати кілька товарів і вказати кількість кожного. (можна просто всі наявні товари)
- Порахувати **підсумкову вартість** замовлення.
- Якщо замовлення перевищує певну суму (наприклад, 1000 грн), запропонувати знижку 10%.

# Що таке масиви?

**Масив** — це набір змінних одного типу, які йдуть підряд у пам’яті, і до яких можна звертатися за номером (індексом).

> Уявіть поличку з багатьма однаковими коробками. В кожній коробці може бути різне значення, але доступ до коробки за номером.

Приклад:
```cpp
int numbers[5]; 
// Це 5 комірок для цілих чисел: numbers[0], numbers[1], numbers[2], numbers[3], numbers[4]
```

---

# Мінімальний приклад використання масиву:
```cpp
#include <iostream>
using namespace std;

int main() {
    const int MAX_PRODUCTS = 5; // максимум 5 товарів
    string names[MAX_PRODUCTS]; // масив назв товарів
    double prices[MAX_PRODUCTS]; // масив цін
    int counts[MAX_PRODUCTS]; // масив кількостей

    int productCount = 0; // скільки товарів ми додали

    // Додавання товару
    cout << "Введіть назву товару: ";
    cin >> names[productCount];

    cout << "Введіть ціну товару: ";
    cin >> prices[productCount];

    cout << "Введіть кількість товару: ";
    cin >> counts[productCount];

    productCount++; // Збільшити кількість доданих товарів

    // Вивід списку товарів
    cout << "\nСписок товарів:\n";
    for (int i = 0; i < productCount; i++) {
        cout << i + 1 << ". " << names[i] << " - " << prices[i] << " грн - " << counts[i] << " шт" << endl;
    }

    return 0;
}
```

---

# Основні моменти, які варто пояснити:

| Що | Як виглядає | Що означає |
|:---|:------------|:-----------|
| Створити масив | `string names[5];` | 5 назв товарів |
| Записати значення | `names[0] = "Хліб";` | перший товар — "Хліб" |
| Зчитати значення | `cin >> names[0];` | з клавіатури в перший елемент |
| Пройти всі товари | `for (int i = 0; i < productCount; i++)` | цикл для виводу всіх товарів |

---

Коли в завданні потрібно додавати кілька товарів — вони просто додають у наступну "комірку" масиву:

```cpp
names[productCount] = введенаНазва;
prices[productCount] = введенаЦіна;
counts[productCount] = введенаКількість;
productCount++;
```

А коли треба вивести — йдуть по всім `i` від 0 до `productCount-1` і показують.

---

# Трошки підказок 
- Масиви починаються **з нуля**.
- Якщо створили `string names[5];`, то можна використовувати `names[0]`, `names[1]`, `names[2]`, `names[3]`, `names[4]`.
- Індекс не повинен виходити за межі (інакше буде помилка).
- Поки що можете працювати з невеликою кількістю товарів (5-10).

# Що таке цикл?

**Цикл** — це коли якась дія повторюється багато разів автоматично.

> Уявіть, що треба 5 разів привітатися. Замість копіювати код 5 разів, можна зробити **цикл**.

---

# Приклад простого циклу for:
```cpp
for (int i = 0; i < 5; i++) {
    cout << "Привіт! Номер " << i << endl;
}
```

**Що тут відбувається:**
- `int i = 0;` — починаємо з 0.
- `i < 5;` — повторюємо, поки `i` менше 5.
- `i++` — після кожного повтору збільшуємо `i` на 1.

---

# Навіщо цикл у програмі?

- Показати **всі товари** в списку.
- Додати **кілька товарів** підряд.
- Зробити **повтор** однієї і тієї самої дії багато разів без копіювання коду.

---

# Мінімальний приклад для товарів:
```cpp
for (int i = 0; i < productCount; i++) {
    cout << i + 1 << ". " << names[i] << " - " << prices[i] << " грн - " << counts[i] << " шт" << endl;
}
```
Тут ми виводимо **всі додані товари** один за одним.
